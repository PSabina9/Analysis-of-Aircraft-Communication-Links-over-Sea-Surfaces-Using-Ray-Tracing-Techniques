Third-year project in the Communication Systems course, focused on modeling and analyzing aircraft communication links over sea surfaces using Ray Tracing techniques in MATLAB.

**Year**: 2025

**Project Overview**:

This project focuses on analyzing the propagation of radio waves for aircraft communication over sea surfaces using Ray Tracing techniques (Shooting and Bouncing Rays) implemented in MATLAB. The study compares theoretical models, such as Free-Space Path Loss (FSPL), with advanced simulations that account for multipath effects caused by reflections from the ocean surface.
The goal is to evaluate signal loss, path gain, and received power under realistic maritime conditions, considering environmental factors like waves, wind, and atmospheric variations.

**Key Highlights**:

Modeling radio wave propagation over dynamic sea surfaces.

Comparison between FSPL and Ray Tracing models.

Simulation parameters include VHF frequency (167.5 MHz), realistic antenna heights, and environmental characteristics.

Evaluation of multipath interference and received signal power.

Provides insights for optimizing antenna placement, transmission power, and link reliability in maritime aviation communications.
